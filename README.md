## 🚀 Featured Projects

### 🌾 FAI — Farm Agent Intelligent System
> Autonomous multi-agent orchestration for precision agriculture

Distributed AI system where specialised agents handle crop monitoring, irrigation scheduling, and yield prediction. Uses PDDL-based planning for goal-driven agent reasoning and constraint satisfaction scheduling for resource management.

- ✅ Multi-agent coordination across 4+ autonomous farm subsystems
- ✅ PDDL planning engine for autonomous decision-making
- ✅ ML-based anomaly detection integrated at sensor level

`Java` `PDDL` `Machine Learning` `Multi-Agent Systems` `CSP`

---

### 🎙️ Speaker Confidence Assessment via NLP
> Explainable AI for human communication analysis

End-to-end NLP pipeline to classify speaker confidence levels from raw text. Combined TF-IDF feature extraction with ensemble classifiers, applied PCA for dimensionality reduction, and surfaced model decisions through SHAP-based explainability.

- ✅ Ensemble model outperformed baseline by ~18% F1 score
- ✅ SHAP explanations for every prediction — fully auditable
- ✅ PCA reduced feature space by 60% with <2% accuracy loss

`Python` `Scikit-learn` `TF-IDF` `SHAP` `PCA` `NLP`

---

### 📄 Resume Analyzer & Job Recommender
> ML-powered career intelligence system

End-to-end ML system that parses raw resume PDFs, extracts structured skill signals using NLP, and predicts candidate job categories. Uses cosine similarity matching and connects to live job APIs for real-time opportunities.

- ✅ NLP extraction pipeline handles unstructured PDF input
- ✅ Classification model predicts job categories with semantic matching
- ✅ Real-time job fetching via external recruitment APIs

`Python` `NLP` `Scikit-learn` `PDF Parsing` `REST APIs`
